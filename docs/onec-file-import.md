# Загрузка счетов из JSON в 1С (обмен через файл)

Платформа выгружает счета в JSON. Эта обработка загружает их в 1С без веб-сервера.

---

## Шаг 1 — Создать внешнюю обработку в 1С

1. Открой **Конфигуратор** → свою базу
2. Меню **Файл** → **Новый** → **Внешняя обработка**
3. В свойствах обработки:
   - **Имя:** `ЗагрузкаСчетовПлатформа`
   - **Синоним:** `Загрузка счетов из платформы`

---

## Шаг 2 — Добавить реквизиты и форму

1. В дереве обработки: **Реквизиты** → Добавить
   - Имя: `ПутьКФайлу`, Тип: `Строка`, Длина: 500
   - Имя: `РезультатЗагрузки`, Тип: `Строка`, Длина: 2000
2. **Формы** → Добавить
   - Имя: `Форма`, Синоним: `Форма`
3. Двойной клик по форме → откроется конструктор формы

---

## Шаг 3 — Элементы формы

На форму добавь:

| Имя элемента | Тип | Привязка |
|--------------|-----|----------|
| `ПолеПуть` | Поле ввода | ПутьКФайлу |
| `КнопкаВыбрать` | Кнопка | — |
| `КнопкаЗагрузить` | Кнопка | — |
| `ПолеСообщение` | Поле ввода, только просмотр | РезультатЗагрузки |

Подписи: «Путь к JSON», «Выбрать файл», «Загрузить в 1С», «Результат».

**Важно:** Поле `ПолеСообщение` должно быть привязано к реквизиту `РезультатЗагрузки`, иначе сообщения не отобразятся.

---

## Шаг 4 — Модуль формы

Открой **Модуль формы** (правая кнопка на форме → Открыть модуль). Вставь:

```bsl
&НаКлиенте
Процедура ВыбратьФайл(Команда)
	Диалог = Новый ДиалогВыбораФайла(РежимДиалогаВыбораФайла.Открытие);
	Диалог.Фильтр = "Файлы JSON (*.json)|*.json";
	Диалог.Заголовок = "Выберите файл выгрузки платформы";
	Оповещение = Новый ОписаниеОповещения("ВыборФайлаЗавершение", ЭтаФорма);
	Диалог.Показать(Оповещение);
КонецПроцедуры

&НаКлиенте
Процедура ВыборФайлаЗавершение(ВыбранныеФайлы, ДополнительныеПараметры) Экспорт
	Если ВыбранныеФайлы <> Неопределено И ВыбранныеФайлы.Количество() > 0 Тогда
		ПутьКФайлу = ВыбранныеФайлы[0];
		ЗагрузитьНаСервере();
	КонецЕсли;
КонецПроцедуры

&НаКлиенте
Процедура Загрузить(Команда)
	Если НЕ ЗначениеЗаполнено(ПутьКФайлу) Тогда
		ПоказатьПредупреждение(, "Укажите путь к файлу JSON");
		Возврат;
	КонецЕсли;
	ЗагрузитьНаСервере();
КонецПроцедуры

&НаСервере
Процедура ЗагрузитьНаСервере()
	РезультатЗагрузки = "Загрузка...";
	Попытка
		ТекстJSON = "";
		ЧтениеФайла = Новый ЧтениеФайла(ПутьКФайлу, КодировкаТекста.UTF8);
		Пока Истина Цикл
			Строка = ЧтениеФайла.ПрочитатьСтроку();
			Если Строка = Неопределено Тогда
				Прервать;
			КонецЕсли;
			ТекстJSON = ТекстJSON + Строка;
		КонецЦикла;
		ЧтениеФайла.Закрыть();
	Исключение
		РезультатЗагрузки = "Ошибка чтения файла: " + ОписаниеОшибки();
		Возврат;
	КонецПопытки;

	Данные = РаботаСJSON.JSONВОбъект(ТекстJSON);
	Если ТипЗнч(Данные) <> Тип("Массив") Тогда
		РезультатЗагрузки = "Файл должен содержать массив JSON";
		Возврат;
	КонецЕсли;

	Создано = 0;
	Ошибки = "";

	Для Каждого Элемент Из Данные Цикл
		Попытка
			Наименование = ?(ЗначениеЗаполнено(Элемент["company"]), Элемент["company"], "Без названия");
			Найденный = Справочники.Контрагенты.НайтиПоНаименованию(Наименование, Истина);
			Если Найденный.Пустая() Тогда
				НовыйКонтр = Справочники.Контрагенты.СоздатьЭлемент();
				НовыйКонтр.Наименование = Наименование;
				НовыйКонтр.ИНН = ?(ЗначениеЗаполнено(Элемент["inn"]), Элемент["inn"], "");
				НовыйКонтр.Записать();
				Найденный = НовыйКонтр.Ссылка;
			КонецЕсли;

			НовыйСчёт = Документы.СчётНаОплату.СоздатьДокумент();
			НовыйСчёт.Дата       = ТекущаяДатаСеанса();
			НовыйСчёт.Контрагент = Найденный;
			НовыйСчёт.Сумма      = ?(ЗначениеЗаполнено(Элемент["amount"]), Элемент["amount"], 0);
			НовыйСчёт.Описание   = ?(ЗначениеЗаполнено(Элемент["description"]), Элемент["description"], "");
			НовыйСчёт.ИдентификаторВнешней = ?(ЗначениеЗаполнено(Элемент["id"]), Элемент["id"], "");
			НовыйСчёт.Статус     = "issued";
			НовыйСчёт.Записать(РежимЗаписиДокумента.Запись);
			Создано = Создано + 1;
		Исключение
			Ошибки = Ошибки + Наименование + ": " + ОписаниеОшибки() + Символы.ПС;
		КонецПопытки;
	КонецЦикла;

	РезультатЗагрузки = "Создано документов: " + Строка(Создано);
	Если ЗначениеЗаполнено(Ошибки) Тогда
		РезультатЗагрузки = РезультатЗагрузки + Символы.ПС + "Ошибки:" + Символы.ПС + Ошибки;
	КонецЕсли;
КонецПроцедуры
```

**Важно:**
- `Диалог.Показать()` вместо `Выбрать()` — иначе ошибка «Использование синхронных методов на клиенте запрещено!»
- Если в документе реквизит называется `ВнешнийИД`, замени `ИдентификаторВнешней` на `ВнешнийИД`

---

## Шаг 5 — Сохранить как EPF

1. **Файл** → **Сохранить как**
2. Сохрани в удобную папку, например: `C:\1c\ЗагрузкаСчетовПлатформа.epf`

---

## Шаг 6 — Использование

1. В платформе gcloude: **Биллинг** → **Выгрузить для 1С** → сохранится `invoices_1c_2026-03-13.json`
2. Запусти **1С:Предприятие** (режим пользователя)
3. Меню **Файл** → **Открыть** → выбери `ЗагрузкаСчетовПлатформа.epf`
4. Нажми **Выбрать файл** → укажи скачанный JSON
5. Нажми **Загрузить в 1С**
6. Документы «Счёт на оплату» появятся в 1С

---

## Формат JSON (для справки)

```json
[
  {
    "id": "uuid",
    "company": "ООО Ромашка",
    "inn": "7712345678",
    "amount": 50000,
    "description": "Хостинг за апрель",
    "created_at": "2026-03-13T..."
  }
]
```

Обработка использует общий модуль `РаботаСJSON` — он должен быть в конфигурации (мы его создавали ранее).
